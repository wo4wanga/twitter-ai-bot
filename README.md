# Twitter AI Bot

一款 Chrome 扩展，自动识别 X (Twitter) 热门推文并通过 AI API 生成高质量回复，帮助提升社交媒体互动效率。

## 功能特点

- **智能识别热门推文**：自动检测点赞/转发数超过阈值的推文
- **AI 流量评估**：从时效性、账号能量、内容价值三个维度评分
- **多语言支持**：支持中文、日文、英文推文过滤
- **内容分类过滤**：可选择 Web3、科技、金融、时事等类别
- **质量阈值控制**：可调节回复质量分数阈值（50-95分）
- **频率限制**：每小时回复上限，避免被封号
- **全自动模式**：自动滚动浏览、检测、回复

## 安装方法

1. 下载本项目代码
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目文件夹

## 使用方法

### 1. 配置 API

在侧边栏中配置你的 AI API：
- **Base URL**：API 端点地址（支持 OpenAI 兼容格式）
- **API Key**：你的 API 密钥
- **模型**：选择要使用的模型

点击「测试 API 连接」验证配置是否正确。

### 2. 设置过滤条件

**语言过滤**：选择要回复的推文语言
- 🇨🇳 中文
- 🇯🇵 日文
- 🇺🇸 英文

**内容类别**：选择感兴趣的内容类别
- Web3/加密货币
- 科技/AI/编程
- 经济/金融
- 时事/新闻
- 游戏/电竞
- 娱乐/明星
- 体育/运动
- 生活/日常

### 3. 调整参数

- **点赞阈值**：推文点赞数达到此值才会触发
- **转发阈值**：推文转发数达到此值才会触发
- **每小时上限**：每小时最多回复数量
- **回复质量阈值**：AI 评分达到此值才会生成回复（50=宽松，80=推荐，95=严格）

### 4. 启动

1. 打开 Twitter/X 网站
2. 点击扩展图标打开侧边栏
3. 点击「一键启动全自动模式」

## AI 评分维度

Bot 会让 AI 从三个维度评估每条推文：

| 维度 | 权重 | 说明 |
|------|------|------|
| 时效动量 | 40% | 30分钟内为黄金期，2小时以上降权 |
| 账号能量 | 20% | 蓝标、大V、高互动量优先 |
| 内容增量 | 40% | 能否引发争议、共鸣或提供干货 |

## 跳过原因

当推文被跳过时，日志会显示具体原因：
- `非目标语言，跳过` - 推文语言不在选择范围内
- `不属于指定类别，跳过` - 推文内容不属于选择的类别
- `评分 XX 分，跳过` - 推文质量评分未达阈值

## 注意事项

- 请合理使用，避免过于频繁的回复导致账号受限
- 建议每小时回复上限设置在 10-15 条
- 回复内容由 AI 生成，请确保符合平台规范
- 首次使用建议先观察 AI 生成的回复质量
